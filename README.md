# 📚 EnglishCard — Веб-приложение для изучения английского языка

Веб-приложение на базе **Python (Streamlit)** и реляционной СУБД **PostgreSQL**, разработанное для эффективного запоминания английских слов. Программа сочетает в себе игровой элемент тестирования, систему персонального ведения словарей и интерактивную визуализацию личной статистики студента.

---

## ✨ Основные возможности (Функционал)

* **🔐 Защищенная авторизация:** Полноценная система регистрации и входа пользователей с безопасным хэшированием паролей по алгоритму SHA-256.
* **📖 Ограниченные раунды обучения:** Обучение разбито на фиксированные сессии по 10 слов. По завершении раунда выводится итоговый экран успеваемости с праздничной анимацией.
* **🧠 Умный выбор слов:** Программа исключает недавние слова из повторения, предотвращая зацикливание, и предлагает 4 случайных варианта перевода на кнопках.
* **⚙️ Персональные словари (Добавление/Удаление):** Пользователь может добавлять свои слова, которые не увидят другие студенты. Также доступна функция «скрытия» (персонального удаления) общих слов.
* **📊 Реальная статистика:** Наглядные метрики точности ответов (`Accuracy`), количества правильных выборов и общего объёма словаря.
* **🎨 Интерактивный дизайн:** Горизонтальные вкладки навигации, звуковые эффекты победы и анимация конфетти при верных ответах.

## 🏛️ Архитектура базы данных (Схема СУБД)

Проект использует реляционную структуру данных из 5 таблиц с настроенными связями «один-ко-многим» (`1:N`) и каскадным удалением зависимых строк:

```text
  +-----------------------------------+

  |               USERS               |  <-- Пользователи системы
  +-----------------------------------+

  | id (PK)           : SERIAL        |
  | username          : VARCHAR(50)   |
  | password_hash     : VARCHAR(255)  |
  +-----------------------------------+

           |                 |
           | (1 : N)         | (1 : N)
           v                 v
  +-----------------+  +-----------------------------------+

  |   USER_STATS    |  |         USER_CUSTOM_WORDS         | <-- Личные слова
  +-----------------+  +-----------------------------------+

  | id (PK) : SERIAL|  | id (PK)           : SERIAL        |
  | user_id : INT   |  | user_id (FK)      : INTEGER       |
  | is_correct: BOOL|  | word_en           : VARCHAR(100)  |
  | answered_at:TS  |  | word_ru           : VARCHAR(100)  |
  +-----------------+  +-----------------------------------+
                             |
                             | (1 : N)
                             v
  +-----------------------------------+      +-----------------------------------+

  |        USER_DELETED_WORDS         |      |           GLOBAL_WORDS            | <-- Базовые слова
  +-----------------------------------+      +-----------------------------------+

  | id (PK)           : SERIAL        |      | id (PK)           : SERIAL        |
  | user_id           : INTEGER       |      | word_en           : VARCHAR(100)  |
  | global_word_id(FK): INTEGER ------------>| word_ru           : VARCHAR(100)  |
  +-----------------------------------+      +-----------------------------------+
```

## 🚀 Инструкция по установке и запуску

### 1. Клонирование репозитория и подготовка окружения
Склонируйте проект с GitHub, создайте и активируйте виртуальное окружение:

```bash
git clone https://github.com
cd bd

python -m venv .venv
# Для Windows:
.venv\Scripts\activate
# Для macOS/Linux:
source .venv/bin/activate
```

### 2. Установка зависимостей
Установите все необходимые библиотеки одной командой:

```bash
pip install streamlit sqlalchemy psycopg2-binary pandas
```

### 3. Настройка подключения к PostgreSQL
Создайте в корне проекта скрытую папку `.streamlit`, а внутри неё — файл конфигурации `secrets.toml`. Заполните доступы к вашей базе данных:

```toml
# Путь к файлу: .streamlit/secrets.toml
[postgres]
host = "localhost"
port = 5432
database = "bd"
user = "postgres"
password = "ваш_надежный_пароль_от_субд"
```
> **Важно:** База данных `bd` должна быть предварительно создана на вашем сервере PostgreSQL. Все таблицы и стартовые 10 слов программа сгенерирует автоматически при первом запуске через встроенную функцию `init_db()`.

### 4. Подготовка медиафайлов
Положите в корневую директорию проекта (рядом с `app.py`) любой аудиофайл длительностью 1-2 секунды, размером не более 100КБ и назовите его **`win.mp3`** для воспроизведения победного звука.
Ограничения по размеру и времени аудио файла позволит избежать подтормаживании веб страницы при каждом правильно ответе.

### 5. Запуск веб-приложения
Запустите локальный сервер Streamlit с помощью команды:

```bash
streamlit run app.py
```
После этого приложение автоматически откроется в вашем браузере по адресу `http://localhost:8501`.
